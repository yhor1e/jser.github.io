2011-11-20のJS: jQuery Mobile 1.0、JavaScript Garden和訳、Sinon.JS

2011年11月20日 Edit on GitHub 編集履歴を見る

JSer.info #44 - jQuery Mobile 1.0がやっと正式リリースされました。
また、JavaScriptにおいてのハマりどころや仕組みなどを上手く紹介してるドキュメントであるJavaScript Gardenの日本語訳が完了し、本家の方にもマージされました。
JavaScriptのテストを補助する機能をいろいろ持ったSinon.JSについての紹介記事が書かれてましたが、このライブラリの著者はテスト駆動JavaScriptの著者でもあります。
JavaScriptでテストが言われ始めるようになって、まだそれほど立ってない気がしますが、いろいろなものがでてきて、ますます活性化しそうな気がします。(両方共リリースされたのは結構前だったりしますが)
