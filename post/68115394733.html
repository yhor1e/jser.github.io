ManningのJavaScript書籍が全て半額のセール中

2013年11月26日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

Manning Publications

Action本等で知られるManning PublicationsもThe Pragmatic Bookshelf と同様の半額のクーポンコードを発行しています。

  • Pragmatic Bookshelfの書籍等が全て半額のセール中 | JSer.info

Manning PublicationsではJavaScript Day!ということで、クーポンコード dotd1125cc を適応する事で、
JavaScriptの書籍限定ですがまだ未発売のMEAPな書籍も対象に50%オフとなってます。

こちらは、EST November 25, 2013 今日限定なのでお早めに選ぶといいかと思います。
該当の書籍はJavaScript Day!に載っています。

クーポンコードの適応方法はJohn Resig: Secrets of the JavaScript Ninja本が50%オフ | JSer.infoが参考になると思います。

来週の月曜日にはApressが全ての書籍が$15になるサイバーマンデーセールを予告しているので、主要な技術書サイトはセールをやる感じだと思います。
